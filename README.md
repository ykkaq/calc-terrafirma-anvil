# TFC Anvil Calculator

TerraFirmaCraft (TFC) の金床加工において、現在値から目標値に到達するための最短手順を計算する静的 Web アプリです。ビルド不要で、ブラウザだけで動作します。

## 機能

- 0 から 150 の範囲内で最短手順を計算
- 金床のルール（最後の操作条件）を最大 3 つまで指定可能
- 連続する同じ操作をまとめて表示し、結果を見やすく簡潔化
- 進捗バーで現在値と目標値を視覚的に確認
- スライダーやキーボード操作で値を直感的に変更

## 使い方

1. **値の入力**: `現在値` と `目標値` を 0〜150 の範囲で入力します。スライダーをドラッグするか、マーカーをクリックしてキーボード（矢印キー・Home・End）で変更できます。
2. **ルールの指定**: 金床 UI に表示されている最後の操作ルールを、必要に応じて最大 3 つまで指定します。
   - 「最後」: 直前の操作
   - 「最後から2番目」: 2つ前の操作
   - 「最後から3番目」: 3つ前の操作
3. **計算実行**: 「計算」ボタンを押すか、入力値を変更すると自動的に計算されます。
4. **手順の実行**: 表示された手順を上から順に金床で実行します。同じ操作が連続する場合は「操作 × 回数」としてまとめて表示されます。

## 操作（1回あたりの変化量）

| 操作 | 変化量 | タイプ |
|------|--------|--------|
| 軽く叩く | -3 | 叩く |
| 普通に叩く | -6 | 叩く |
| 強く叩く | -9 | 叩く |
| 引き延ばす | -15 | 引き延ばす |
| 打ち抜く | +2 | 打ち抜く |
| 曲げる | +7 | 曲げる |
| 据え込む | +13 | 据え込む |
| 縮める | +16 | 縮める |

## 計算方法

幅優先探索（BFS）を用いて最短手順を求めます。

- 状態: 現在の値（0〜150）と直近 3 つの操作履歴
- 遷移: 各操作を適用し、値が範囲内に収まる場合のみ遷移
- 終了条件: 目標値に到達し、かつ指定したルールをすべて満たす
- 同一状態の重複探索を回避するため、状態をキーとして記録

## GitHub Pages での公開

このリポジトリはビルド不要です。GitHub Pages の設定を以下のようにしてください。

1. リポジトリの Settings > Pages を開く
2. Source を「Deploy from a branch」に設定
3. Branch で対象ブランチ（例: `main`）とフォルダ `/root` を選択
4. Save をクリック

数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセスできます。
