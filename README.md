# TFC Anvil Calculator

TerraFirmaCraft の金床加工について、現在値から目標値までの最小手順を計算する静的 Web アプリです。

## 使い方

1. `現在値` と `目標値` を 0..150 で入力します。
2. 金床 UI に表示された最後のルールを最大 3 つ指定します。
3. 表示された手順を上から順に実行します。

## GitHub Pages

このリポジトリはビルド不要です。GitHub Pages の Source を `Deploy from a branch` にして、対象ブランチの `/root` を公開してください。

## 計算

操作量は以下として扱います。

- 軽く叩く: -3
- 普通に叩く: -6
- 強く叩く: -9
- 引き延ばす: -15
- 打ち抜く: +2
- 曲げる: +7
- 据え込む: +13
- 縮める: +16

途中状態が 0..150 の範囲を外れる手順は除外し、幅優先探索で最短手順を求めます。
